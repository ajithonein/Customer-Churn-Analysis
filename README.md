# Customer Churn Analysis

This project predicts customer churn behavior and identifies key factors influencing customer retention.

## Objectives
- Identify churn patterns
- Analyze customer demographics
- Detect high-risk churn customers
- Suggest retention strategies

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Decision Tree

## Key Insights
- Contract type impacts churn significantly
- Monthly charges influence churn probability
